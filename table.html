Terms Definitions
cd Allows you to change the directory you are in.
pwd Allows you to check which directory you are currently in.
mkdir Allows you to make a new directory.
nano Allows you to make a new file and write in it.
cp -r Allows you to recursevly copy a folder and everything inside that folder.
rm -fr Allows you to forcefully remove a folder and everything in it.
sudo "Superuser do" Allows you to execute commands without having to know the root password.
